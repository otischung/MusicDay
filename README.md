## Usage

1. Install Linux requirements

   ```bash
   ./linux_install_requirements.sh
   ```

   

2. Install Python requirements

   ```
   pip install -r requirements.txt
   ```

   

3. Execute by the following command:

   ```bash
   python index.py
   ```
